# Bitácora de cambios

Este documento registra los avances principales del Banco Digital de Lengua de Señas Peruana.

La bitácora permite dejar evidencia del desarrollo progresivo del proyecto, facilitar la revisión institucional y diferenciar la versión original de futuras mejoras o adaptaciones.

## Versión inicial funcional

Se crea el Banco Digital de Lengua de Señas Peruana como aplicación web educativa de consulta visual, organizada por categorías y con búsqueda por palabra.

Se incorpora una estructura básica compuesta por archivos HTML, CSS, JavaScript, datos en formato JSON e imágenes organizadas por categorías.

Se reconoce como fuente del material visual de referencia la Guía de Lengua de Señas Peruana del Ministerio de Educación del Perú.

Se registra la autoría del desarrollo, organización, clasificación, adaptación digital y estructura de consulta a cargo de Gabriel Berrospi, con apoyo institucional del CREBE "Señor de los Milagros" - Ucayali.

## Incorporación de respaldo documental

Se actualiza el archivo `README.md` para incluir la carpeta `docs`, el archivo `LICENSE` y una explicación más clara sobre el respaldo pedagógico, autoral, institucional y organizativo del proyecto.

Se incorpora el archivo `LICENSE`, con aclaración de que la licencia corresponde al código del repositorio y no otorga automáticamente derechos sobre imágenes, logos institucionales, documentos externos o recursos de terceros.

Se agregan documentos de respaldo en la carpeta `docs`:

- `alcance-pedagogico.md`
- `fuentes-y-creditos.md`
- `uso-permitido.md`
- `respaldo-institucional.md`
- `estructura-del-proyecto.md`
- `bitacora-de-cambios.md`

Cada mejora significativa debe anotarse con fecha, descripción breve y motivo del cambio.

Formato sugerido:

```text

Cambio realizado:

```

Este registro no reemplaza los commits de GitHub, pero ayuda a documentar el sentido pedagógico, autoral e institucional de cada modificación importante.
