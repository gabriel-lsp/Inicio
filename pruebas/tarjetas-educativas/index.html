Cómo usar la práctica
  • Use Anterior y Siguiente para recorrer las tarjetas.
  • Presione Espacio o Enter sobre la tarjeta para mostrar u ocultar el contenido.
  • Use las flechas izquierda y derecha para cambiar de tarjeta.
  • La lectura en voz alta utiliza las voces disponibles en su navegador.
  • “Imprimir tarjeta” también permite guardarla como PDF desde el cuadro de impresión.
